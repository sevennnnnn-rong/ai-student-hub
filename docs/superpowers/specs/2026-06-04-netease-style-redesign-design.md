# AI 学生Hub — 网易云风格 UI 重设计

**日期**: 2026-06-04
**状态**: 已批准
**范围**: 前端全量 UI 重设计（主题 + 布局 + 组件 + 弹窗系统）

---

## 1. 目标

将前端 UI 从当前的 "Liquid Glass" (Apple visionOS) 风格迁移为网易云音乐风格，同时支持深色/浅色双主题，并在 PC 端增加弹窗化功能（番茄钟、笔记、AI对话），让用户学习时不打断主流程。

## 4. 弹窗系统（PC 端）

三个核心学习功能支持弹窗化，使用 Tauri 2.x 多窗口 API 实现原生浮动窗口。

### 弹窗类型

| 弹窗 | 尺寸 | 特性 |
|------|------|------|
| 番茄钟 | 240×280px | always_on_top, 无标题栏, 进度条, 暂停/跳过按钮 |
| 快速笔记 | 280×320px | always_on_top, 自动保存, 标签分类 |
| AI 对话 | 300×360px | always_on_top, 流式输出, 消息列表 |

### 弹窗设计规范

- 标题栏: 渐变红色背景 (`#C20C0C` → `#9B0909`)
- 标题栏按钮: 最小化(—)、最大化(□)、关闭(✕)，圆形半透明背景
- 内容区: `--bg-elevated` 背景
- 圆角: 12px
- 阴影: `0 8px 32px rgba(0,0,0,0.4)`
- 可拖拽移动

### 响应式策略

| 视口 | 策略 |
|------|------|
| >768px | 浮动弹窗（Tauri 多窗口） |
| 480-768px | 半屏 sheet 或侧滑面板 |
| <480px | 全屏页面替换 |

## 5. 组件规范

### 按钮

- 圆角: 20px（全圆角）
- 主按钮: `--accent` 填充 + 白色文字
- 次要按钮: `rgba(255,255,255,0.08)` 填充 + 浅灰文字
- 悬停: 颜色加深 15%
- 过渡: 0.15s ease

### 卡片

- 圆角: 12px
- 背景: `rgba(255,255,255,0.03)` (深色) / `#fff` (浅色)
- 边框: 1px `--border`
- 悬停: 背景色加深 + 微上浮 `translateY(-2px)`

### 列表项

- 高度: 56-64px
- 分割线: 底部 1px `--border`
- 圆点标记: 8px 圆形，颜色表示优先级（红/黄/白）

### 滚动条

- 宽度: 5px（全局默认）
- 轨道: transparent
- 滑块: `rgba(194,12,12,0.3)`，悬停加深到 `0.5`
- 圆角: 10px

### 统计卡片

- 三列网格布局
- 标签: 11px `--text-muted`
- 数值: 22px 粗体，品牌红/白
- 趋势: 11px，白色

### 标签

- 圆角: 6px
- padding: 3px 8px
- 字号: 10px
- 活跃: 红底红字
- 待定: 灰底灰字
